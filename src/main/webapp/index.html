Git SHA-1 Demo

A demo of how a Git SHA-1 can be embedded in a project:

  • Added to the DOM:
  • View the source of this page to see it added to the DOM as a html comment at the bottom of this page
  • In the buildNumber.properties file (look in the WEB-INF/classes folder in the generated .war file)
  • As an ordinary Java method call to the PropertyReader.getGitSha1()
  • As a JSON response from the server: /build-number/git-sha-1
  • As an entry in the MANIFEST.MF (look in the META-INF folder in the generated .war file)
For more information, read the blog post at http://blog.jayway.com/2012/04/07/continuous-deployment-versioning-and-git
